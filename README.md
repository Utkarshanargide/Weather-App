# 🌦 Weather App

A Java Servlet & JSP based Weather Application that fetches real-time weather information using the OpenWeather API.

---

## 🚀 Features

- Search weather by city
- Temperature
- Humidity
- Wind Speed
- Pressure
- Weather Icon
- Responsive UI

---

## 🛠 Technologies

- Java 17
- Maven
- Servlet
- JSP
- HTML
- CSS
- JavaScript
- Apache Tomcat 11
- Git
- GitHub
